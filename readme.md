# Image Search App

A simple and responsive Image Search Application built using HTML, CSS, and JavaScript. The app allows users to search for images using an API and view results instantly. Users can also load more images with the "Show More" button.

## Features

* Search images by keyword
* Real-time image fetching using API
* Responsive user interface
* Click on any image to open the original source in a new tab
* Show More button for pagination
* Clean and user-friendly design

## How It Works

1. User enters a search keyword.
2. The application sends a request to the image API.
3. Images are fetched and displayed dynamically.
4. Clicking an image opens its original page.
5. The "Show More" button loads additional results.
